Narrono Game Manual

Quickstart Guide

Welcome to the Narrono Quick Start materials! This collection is designed to help new players and Game Masters jump into the Narrono roleplaying system with minimal preparation. Whether you're new to tabletop RPGs or an experienced player trying Narrono for the first time, these materials will get you playing quickly.

What's Included

Core Materials

  • Quick Rules: A streamlined version of the Narrono rules with just what you need to start playing

Reference Guides

  • Reference Guides: Quick reference guides for key game mechanics
    • Resolution Guide: Step-by-step guide to resolving actions
    • Asset Guide: How to use and manage character assets
    • Relationship Reference: Understanding character relationships

How to Use These Materials

For Players

  1. Read the Quick Rules document to understand the basic mechanics
  2. Choose a pre-generated character from one of the modules
  3. Familiarize yourself with your character's traits, assets, and relationships
  4. Review the reference guides for any mechanics you're unsure about
  5. Jump into the adventure!

For Game Masters

  1. Read the Quick Rules document to understand the basic mechanics
  2. Choose a module that interests your group
  3. Review the module's setting and story documents
  4. Familiarize yourself with all pre-generated characters
  5. Read through the reference guides to prepare for common questions
  6. Guide your players through the adventure, adapting as needed

Getting Started

The quickest way to begin is with the Shadow Court module:

  1. Have each player select one of the pre-generated characters
  2. Read the setting document together to establish the world
  3. Begin with Scene 1 from the story document
  4. Use the Pool System to resolve actions as described in the Quick Rules
  5. Let the story evolve based on player choices and dice outcomes

Moving to the Full System

Once you're comfortable with the quick start rules, you can explore the full Narrono system:

  1. Read the complete rulebook for detailed mechanics and options
  2. Create your own characters using the full character creation system
  3. Build custom settings using the Five Axes framework
  4. Develop your own assets and relationships
  5. Explore advanced mechanics and optional rules

Happy gaming!
