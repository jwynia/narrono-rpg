Narrono Game Manual

Playtest Materials

This section contains materials useful for playtesting the Narrono game system. These materials are designed to help Game Facilitators (GFs) and players test the game system and provide structured feedback.

Contents

Relationship Guides

Tools for creating and managing character relationships:

  • Character-to-Character Relationships: Relationship types, status establishment, tension points, development potential
  • Character-to-Faction Relationships: Position establishment, influence mapping, obligation framework, benefit structure
  • Relationship Web Construction: Network visualization, connection density, tension distribution, balance assessment

Session Templates

Structured forms for different session types:

  • First Session Template: Character introduction, setting establishment, Relationship Web creation, initial goal setting
  • Standard Session Template: Scene planning, character spotlight tracking, relationship development, asset management
  • Milestone Session Template: Character development, setting evolution, relationship transformation, goal reassessment

GF Screens

Printable reference panels for Game Facilitators:

  • Core Rules Panel: Resolution methods, outcome interpretation, asset states, relationship effects, and Shared Story Pool management
  • Setting Management Panel: Five Axes framework, setting evolution, faction tracking, location management, and axis shift implementation
  • Character Management Panel: Trait development, asset recovery, relationship evolution, character goal tracking, and group dynamics
  • Session Management Panel: Scene creation, pacing guidelines, Compensation Principle, Dynamic Outcome examples, and troubleshooting

How to Use These Materials

Exporting to PDF

All materials can be exported to PDF format for printing:

  1. Build the mdBook with PDF output enabled
  2. Access individual PDF files from the build directory
  3. Print materials as needed for table reference

Printing Recommendations

  • Print GF screens double-sided if possible
  • Use card stock or laminate for durability
  • Consider color printing for better readability
  • Print multiple copies of session templates for repeated use

Feedback Collection

When using these materials for playtesting:

  1. Use the playtest tracking template in the simulations/templates directory
  2. Record observations about usability and clarity
  3. Note any missing information or confusing elements
  4. Document how the materials affected gameplay
  5. Suggest improvements or additions

Playtest Methodology

Preparation

  1. Select appropriate materials based on playtest focus
  2. Distribute templates and reference materials to participants
  3. Establish feedback collection methods
  4. Set clear goals for the playtest session

Execution

  1. Use the materials in live play situations
  2. Document observations during play
  3. Note any confusion or questions about the materials
  4. Track how materials influence the play experience

Analysis

  1. Collect all feedback after the session
  2. Compile observations and suggestions
  3. Identify patterns and common issues
  4. Prioritize improvements based on impact

Iteration

  1. Update materials based on feedback
  2. Test revised versions in subsequent sessions
  3. Compare results against previous versions
  4. Continue refinement until materials meet objectives

These playtest materials implement the systems described in the Narrono RPG core rules, providing practical tools for testing and feedback collection.
